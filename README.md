# Trading Agent Dashboard

A React paper-trading dashboard for experimenting with model-driven market analysis, simulated execution, profit/loss
tracking, and backtesting.

## Features

- Simulated live market feed for crypto and equity symbols
- Selectable trading models: momentum, mean reversion, risk-balanced, and breakout
- Start/pause/step controls for paper trading
- Portfolio equity, cash, open P/L, fees, position, and trade log views
- Risk controls for allocation, stop loss, and take profit
- Deterministic backtest runner with final equity, return, win rate, drawdown, and equity curve

> This project is an educational simulator. It does not connect to a real broker or exchange, and profit is not
> guaranteed.

## Getting started

```bash
npm install
npm run dev
```
