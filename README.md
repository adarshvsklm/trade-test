# Trading Agent Simulator

A React-based trading simulator that supports:

- real-time market analysis with multiple strategy models,
- simulated order execution and portfolio tracking,
- live P/L, drawdown, and trade history visibility,
- model selection (ensemble voting), and
- full-history backtesting with performance metrics.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This project is a simulation tool. It does **not** connect to real exchanges.
- Profitability in synthetic or backtest data does not guarantee real-world returns.
