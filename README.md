# Trading Agent Dashboard (React + Node.js)

This project provides a **simulated AI trading agent** with:

- Live strategy execution on synthetic market data
- Model selection (Momentum, Mean Reversion, Breakout)
- Start/stop trading controls
- Real-time portfolio and risk metrics
- Trade history and equity curve visualizations
- Backtesting panel with configurable inputs

> Important: This app is for simulation and education. It cannot guarantee profits in real markets.

## Tech Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Monorepo:** npm workspaces

## Run Locally

```bash
npm install
npm run dev
```

- Frontend: `http://localhost:5173`
- Backend API: `http://localhost:4000`

## API Overview

- `GET /api/models` - available strategy models
- `GET /api/state` - current live trading state
- `POST /api/trading/start` - start simulated live trading
- `POST /api/trading/stop` - stop trading
- `POST /api/backtest/run` - run backtest

## Trading Engine Notes

- Uses synthetic candles generated by a market simulator
- Applies model signal logic to open/flip long-short positions
- Tracks:
  - realized/unrealized/total P&L
  - win rate and drawdown
  - sharpe-style score from equity returns
- Includes configurable stop-loss and take-profit controls
