# Trade Test

A React trading-agent dashboard for exploring model selection, paper trading, profit/loss tracking, and backtesting.

## Features

- Market selector for crypto and equity examples.
- Model picker for momentum, mean-reversion, risk-filter, and breakout strategies.
- Paper trading controls to start or pause the simulated agent.
- Live signal panel with price, conviction, rationale, momentum, RSI, and volatility.
- Portfolio view with cash, open position, equity, and open P/L.
- Backtest replay with net P/L, return, win rate, drawdown, trade markers, and an equity sparkline.

> This project is a simulation and product prototype. It does not connect to a broker and cannot guarantee profit.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
